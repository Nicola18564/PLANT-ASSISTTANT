# MediAssist OpenEnv

A unified AI healthcare decision and monitoring environment built in Python.

## Features

- Symptom-based disease detection
- Step-by-step patient guidance
- Medication suggestion engine
- Doctor recommendation helper
- Risk prediction using age, BMI, and risk factors
- Mental health support guidance
- Fitness and preventive care plans
- Health monitoring alerts from vitals
- Scenario support for rural healthcare, accessibility, epidemic cases, and elderly fall detection
- Simulated voice-based assistant via text input

## Getting Started

Run the interactive environment with:

```bash
python -m mediassist_openenv
```

or from the repository root:

```bash
python main.py
```

## Project Structure

- `mediassist_openenv/core.py`
- `mediassist_openenv/support.py`
- `mediassist_openenv/scenarios.py`
- `mediassist_openenv/app.py`
- `mediassist_openenv/utils.py`
