# PLANT ASSISTTANT
This project is a Python-based weather notification system designed for smart agriculture and plant care. It uses the OpenWeatherMap API to fetch real-time weather data and future forecasts for Bengaluru. The system analyzes weather conditions such as temperature, humidity, and climate status, and provides desktop notifications on the laptop screen to alert users about upcoming weather changes.  The project uses Python modules like requests, datetime, smtplib, email.mime, and plyer to retrieve weather information, process time data, and display alerts. This system can help farmers or plant caretakers monitor weather conditions and take timely actions to protect plants.  Technologies Used:  Python  OpenWeatherMap API  Plyer (Desktop Notifications)  Requests Library
